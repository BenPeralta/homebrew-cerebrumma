# Homebrew Tap for Cerebrumma

Portable, git-tracked AI Brain for developers. One Brain. Any Tool.

## Install

```sh
brew install BenPeralta/cerebrumma/cerebrumma
```

Or tap first, then install:

```sh
brew tap BenPeralta/cerebrumma
brew install cerebrumma
```

This installs the `cerebrum` CLI. To also set up the MCP server and
auto-configure Claude Code, use the full installer:

```sh
curl -fsSL https://get.cerebrumma.com | sh
```
